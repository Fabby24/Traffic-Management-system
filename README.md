# Studio Traffic Management System

A web-based **Studio Traffic Management System** designed to help creative agencies efficiently manage projects, tasks, team workloads, and deadlines through a centralized platform.

##  Current Progress

### Backend Authentication Module

The following features have been implemented:

- User Registration
- User Login
- JWT Authentication
- Role-Based Access Control (Admin & Team Member)
- Password Hashing with bcrypt
- Forgot Password
- Reset Password
- User Profile Management
- Protected Routes Middleware
- Input Validation
- Centralized Error Handling
- MySQL Database Integration
- MVC Project Architecture
- API Testing with Postman

##  Tech Stack

- **Backend:** Node.js, Express.js
- **Database:** MySQL
- **Authentication:** JWT & bcrypt
- **API Testing:** Postman
